Ce site a pour but de rassembler toutes les étapes servant à la création d'un dépot sous GitHub, et à la gestion d'un fichier html/css par un groupe de plusieurs personnes en simultané.

Récupération des fichiers à nouveau:

- Vérifier que le dossier est propre : git status
- Sinon effectuer un git add NomDuFichier suivi
- git remote add upstream:
https://github.com/WyrwinskiLeo/TP4Revue.git
- Enfin effecuter un git pull upstream master

  • Une fois le dépot propre:
    • pour changer de branche : checkout master
    • git push -all upstream master
